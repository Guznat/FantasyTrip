Tutorial 0 : Simple steps to make camshake working

First step (create objects)
1. Create an empty scene.
2. Create an empty GameObject at position (0,0,-10) and name it as "CamRoot".
3. Create an empty GameObject, drag it to CamRoot, set its relative position to (0,0,0) and rename it to "Offset1".
4. Drag the "Main Camera" to "Offset1".
5. Create an cube from menu GameObject->Create Other->Cube at position (0,0,0). This is a reference object to let us know how the camera is shaking.

Second step (attach script and adjust params)
1. Attach WHCameraShake.cs to "Offset1".
   If you want to test the shaking effects now, just play the game in editor, then disable and enable the WHCameraShake component of object "Offset1" in inspector window.
   WHCameraShake has a default vertical shaking config (as you see in the Pos Speed section in the screen shot below).
   So you will see the cube is shaking vertically (actually it is because the "Main Camera" is shaking).


2. Set Pos Speed to (100,100,0), then test again (by disabling and enabling the WHCameraShake component ), you will see the camera is shaking in the new direction you just specified. You can set Pos Speed to any value you like when game is running and test to see the new shaking effects.

Third step (random shaking)
1. Attach WHCameraShake.cs to "Main Camera".
2. Attach WHCameraRandomShaker.cs to "CamRoot".
3. Attach tst0.cs to "CamRoot".


4. Set Pos Speed of WHCameraShake in "Main Camera" to (100,0,0) and set Pos Speed of WHCameraShake in "Offset1" to (0,100,0).


5. Run the game. You will find camera shaking in random directions when you left click on scene.
   This random effects is made by combining 2 one direction (random speed) shaking movements togather.

Fourth step (making a little swirl)
1. Set delay of WHCameraShake in "Main Camera" to 0.05


2. Run the game. You will find camera shaking in a random ellipse trail.
   The reason is simple: the delay of 0.05 second makes a moving phase difference.

You can find this sample in Assets/CamShake/Scene/Tst0 ****
